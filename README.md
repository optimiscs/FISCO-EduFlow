# 学链通区块链浏览器

## 项目简介

学链通是基于fisco区块链技术的学历全流程管理系统，立足于 **区块链技术** 赋能教育新基建的国家战略方向，应用于 **跨境学历互认、政企人才快速核验、个人教育数据主权管理** 等高价值场景，致力于保障人才流动中的**可信认证与隐私安全**，能够提供学历信息上链、证书验证、企业验证等功能。除此之外，区块链浏览器是系统的重要组成部分，用于查看和监控区块链网络的实时数据。

## 核心技术
区块链架构设计采用双链混合模式，设计三类区块链通道，保障系统同时能够在不需要每个参与节点验证网络中的每笔交易的情况下工作，并且不失去网络对每笔交易都是有效的信任。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_11](https://github.com/user-attachments/assets/18388299-c11f-4e66-a3f3-8d6997bca32c)

区块链的所有节点执行相同的计算、存储相同的数据，这样一种冗余的架构在保障了安全性和非中心化的同时，也导致区块链处理交易的速度非常慢，且费用昂贵。于是本作品在采取主链 + 侧链的混合模式的基础上，将小额多次的认证收费、数字人民币交易的合约部署到状态通道上，其中利用状态通道交易的流程为：

(1)交易方在链上通过浏览器钱包锁定一定量的资产，在区块链上记录并开辟状态通道;

(2)在通道内进行相互交易和状态更新,但不提交到链上;

当任一方想要关闭通道时，提交最终状态到区块链上进行清算。若另一方有异议，可在规定时间内申请链上仲裁。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_12](https://github.com/user-attachments/assets/05d53202-09b1-4de8-9b71-e5ba7229feb0)

由于联盟主链与联盟侧链是两条权限不同的区块链，所以链上数据必须同步，这一步在验证信息真伪上也是重要的。本作品设计了主侧链数据同步、数据验证的智能合约，因此使用到了Merkle Tree 算法进行哈希锁定。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_13](https://github.com/user-attachments/assets/7c7e6a63-766e-446f-b153-4e95884bcce1)


我们根据学位、学历认证智能合约模型，结合传统学位学历认证流程，提出使用学位学历认证智能合约算法，此认证模型实现了系统的去中心化，交易和合约信息公开、透明、不可篡改，合约自动存储和执行。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_14](https://github.com/user-attachments/assets/b4c0a543-ce95-41ff-8e7a-ed9638502910)

针对作品简介中提到的学历全流程管理认证问题和我们对国内外研究现状的调研，我们决定采用“主链+侧链”的双链混合模式，结合“链下状态通道”，设计三类区块链通道，下图是我们的设计大纲：

智能合约模块可以起到满足用户与认证申请方之间的信任，以及数据不可篡改的作用：
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_17](https://github.com/user-attachments/assets/a2999d1d-00f3-461f-9832-6b6213038af8)


共识机制通过各类共识算法来实现分布式节点间数据的一致性和真实性。在本系统中采用一种基于移动设备的高吞吐量共识协议，仅需智能手机即可参与区块验证和共识，鼓励大量参与者参与区块的产生。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_18](https://github.com/user-attachments/assets/d7296847-5a14-47c1-ab9c-1bcc1047accd)

考虑到节点安全性和电子认证的缺陷，我们决定引入CA认证保障安全性,提高防伪性能。CA认证，即电子认证服务，是指为电子签名相关各方提供真实性、可靠性验证的活动。
![【演示文档-PPT】学链通—基于 FISCO 区块链的学历全流程管理系统_19](https://github.com/user-attachments/assets/6870e36a-e08c-466e-a246-390a44b62b40)


零知识证明技术可以有效保护用户的隐私安全信息，即可让验证者相信他们是正确的。在系统中，我们采用零知识证明保证学生的“非必要隐私信息”，防止隐私的“过度泄露”。

## 主要功能

- **区块链概览**：显示区块高度、交易速度、节点数量和总交易数等关键指标
- **区块查询**：查看最新区块信息，包括区块高度、生成时间、交易数和区块哈希
- **交易查询**：查看最新交易信息，支持按类型筛选（学历上链、证书验证等）
- **数据统计**：直观展示交易数量趋势和节点类型分布
- **节点监控**：实时监控各节点状态、资源使用率和同步情况
- **学历验证**：支持证书编号、个人信息、扫码三种验证方式
- **证书管理**：学校可创建、发布、撤销证书，学生可查看自己的证书

## 系统架构

学链通系统包含以下模块：
- **前端**：HTML5, CSS3, JavaScript实现的用户界面
- **后端**：基于Node.js和Express的API服务
- **数据库**：MongoDB存储用户、证书、验证记录等信息
- **区块链**：FISCO BCOS区块链网络，存储证书哈希和验证信息
- **智能合约**：实现证书发布、验证、撤销等业务逻辑

## API接口

### 用户认证
- `POST /api/auth/register`: 用户注册
- `POST /api/auth/login`: 用户登录
- `GET /api/auth/me`: 获取当前用户信息

### 区块链浏览
- `GET /api/blockchain/overview`: 获取区块链概览
- `GET /api/blockchain/blocks`: 获取最新区块列表
- `GET /api/blockchain/transactions`: 获取最新交易列表

### 证书管理
- `POST /api/certificates`: 创建证书
- `GET /api/certificates`: 获取证书列表

### 验证服务
- `POST /api/verify/certificate-number`: 通过证书编号验证
- `POST /api/verify/personal-info`: 通过个人信息验证
- `GET /api/verify/history`: 获取验证历史记录
